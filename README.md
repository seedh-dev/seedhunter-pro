# SeedHunter Pro

Advanced multi-chain cryptocurrency seed phrase recovery suite.

## Supported Chains (10)
- Bitcoin (BTC) — bc1q Bech32
- Ethereum (ETH) — 0x EIP-55
- Solana (SOL) — Base58

## Features
- BIP39/BIP32/BIP44 compliant
- Multi-chain parallel scanning
- Provider fallback system
- Dark theme
