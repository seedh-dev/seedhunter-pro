# Supported Chains

## Bitcoin (BTC)
- Address format: bc1q... (Bech32 SegWit)
- Derivation: m/84'/0'/0'/0/0
- Balance API: mempool.space, Blockstream

## Ethereum (ETH)
- Address format: 0x... (EIP-55)
- Derivation: m/44'/60'/0'/0/0
- Balance API: public RPC endpoints

## Solana (SOL)
- Address format: Base58 (Ed25519)
- Derivation: m/44'/501'/0'/0'
- Balance API: Solana public RPC
